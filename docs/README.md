# zAgent.ps

A modern "Coming Soon" landing page for zAgent, featuring a minimalist code editor design with an animated neural network background.

## Features

- **Code Editor Aesthetic**: Clean, minimalist design inspired by code editors with syntax highlighting
- **Animated Background**: Interactive neural network visualization with connecting nodes and pulses
- **Responsive Design**: Fully responsive layout that works on all devices
- **RTL Support**: Right-to-left layout optimized for Arabic content
- **Typing Animation**: Dynamic text typing effect with multiple phrases
- **Social Links**: Quick access to Instagram, Facebook, WhatsApp Business, and GitHub

## Technologies

- Pure HTML, CSS, and JavaScript
- Canvas API for neural network animation
- Google Fonts (Cairo)
- No dependencies or build tools required

## Files

- `index.html` - Main HTML file with embedded styles and scripts
- `zAgent.mp4` - Logo animation video
- `zAgent-favicon.png` - Favicon and app icon
- `zAgent-app.png` - App icon image
- `Screenshot 2025-11-07 at 1.39.39 PM.png` - Screenshot

## Usage

Simply open `index.html` in a web browser. No build process or server required.
